# Diplom_3

Папка data:

- data.py содержит данные об ингредиентах
- urls.py содержит ссылку на сатй StellarBurgers

Папка helper:

- helpers.py содержит функцию создания уникальных данных для пользователя

Папка locators:
- account_page_locators cодержит локаторы элементов страницы авторизации и регистрации пользователя.

- main_page_locators cодержит локаторы элементов главной страницы приложения, конструктора бургеров и модальных окон.

- order_page_locators cодержит локаторы элементов страницы «Лента заказов», включая номера заказов, счётчики и список заказов в работе.


Папка pages:

- base_page содержит базовые методы 
    поиск элементов;
    ожидания;
    клики;
    ввод текста;
    получение текста;
    проверка отображения элементов;
    скролл;
    drag-and-drop;
    ожидание исчезновения элементов.

- main_page содержит методы для работы с главной страницей приложения:

    переход в раздел «Конструктор»;
    переход в раздел «Лента заказов»;
    открытие модального окна ингредиента;
    работа с ингредиентами;
    проверка отображения модальных окон;
    перетаскивание ингредиентов в корзину;
    работа со счётчиками ингредиентов.

Папка tests:
- OrderFeedPage cодержит методы для работы с разделом «Лента заказов»:

    регистрация и авторизация пользователя;
    создание заказа;
    получение номера заказа;
    закрытие модального окна заказа;
    получение счётчиков заказов;
    проверка отображения заказа в разделе «В работе»;
    проверка увеличения счётчиков заказов.

- conftest.py содержит в себе фикстуры открытия и закрытия браузера Chrome и Firefox
- test_main_page содержит тесты на следующие проверки:
    переход по клику на «Конструктор»;
    переход по клику на раздел «Лента заказов»;
    если кликнуть на ингредиент, появится всплывающее окно с деталями;
    всплывающее окно закрывается кликом по крестику;
    при добавлении ингредиента в заказ счётчик этого ингредиента увеличивается.


- test_order_feed_page содержит тесты на следующие проверки:
    при создании нового заказа счётчик «Выполнено за всё время» увеличивается;
    при создании нового заказа счётчик «Выполнено за сегодня» увеличивается;
    после оформления заказа его номер появляется в разделе «В работе».
